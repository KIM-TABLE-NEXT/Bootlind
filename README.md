## 프로젝트 회고
>이전에 올린 게시글 처럼 회원탈퇴 로직에 관하여 고민해보면서 로직만으로도 시스템의 성능을 향상 시킬 수 있다는 점을 깨달아서 좋았고, 백지상태였던 프론트엔드를 맨땅에 헤딩하며 공부하고 구현해보면서 `HTML`과 `Javascript`를 어느정도는 다룰 수 있게되어 매우 만족스럽다.
다음 프로젝트는 프론트엔드는 물론, `AWS`와 `소셜로그인` 까지 적용해보고 싶다.
